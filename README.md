# AI-Scam-Detector-extension

AI Scam Detector is a browser extension designed to block phishing URLs. By leveraging a backend API and a phishing dataset, this extension ensures a safer browsing experience by dynamically identifying and blocking malicious links.

## Features

- **URL Blocking:** Prevents access to URLs listed in the `cleaned_phishing_dataset.csv` file.
- **Dynamic Updates:** Utilizes a backend API to keep the phishing URL list updated.
- **Manifest V3 Support:** Fully compatible with the latest Chrome extension standards.

## Prerequisites

Before setting up the project, ensure you have the following installed:

- Node.js and npm
- Python
- venv
